flash的卸载
macOS
Blog
发布日期:   2021-02-09
文章字数:   296
阅读时长:   1 分

起因

Chrome 和 Firefox 全面禁用 Adobe Flash，导致单位的办公系统无法使用上传功能。在网管人员的「建议」之下，安装了因国内某公司提供服务的「特供版」Flash，确实可用。出于心理洁癖，不信赖国产厂商，所以想卸载这个 APP，但在 AppCleaner 和 CleanMyMac 的软件列表中均未找到它，在/Application 文件夹中也未找到相应文件，所以才有本文。

解决方案

以下内容来自：在 macOS 中卸载 Flash Player，可直接查看 Adobe 官网，获取最新内容。

  1. 下载 Adobe Flash Player 卸载程序

    • Mac OS X 版本 10.6 及更高版本：uninstall_flash_player_osx.dmg
    • Mac OS X 版本 10.4 和 10.5：uninstall_flash_player_osx.dmg

  2. 双击打开所下载的「卸载程序」，按照提示运行至卸载完成

  3. 清理卸载残余垃圾

    • <个人目录>/资源库/Preferences/Macromedia/Flash\ Player
    • <个人目录>/资源库/Caches/Adobe/Flash\ Player

  4. 卸载完成
